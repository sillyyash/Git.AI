================================================================================
                          QUICK START
================================================================================

1. Import the planner:
   from agents.planner import create_planner

2. Create a planner instance:
   planner = create_planner(debug=False)

3. Create a plan:
   plan = planner.plan(
       "rename the login function to authenticate",
       repository_index,
       dependency_graph,
       relationship_graph,
   )

4. Access plan components:
   - plan.intent          Intent classification
   - plan.risk            Risk level assessment
   - plan.complexity      Complexity level
   - plan.affected_files  Files to modify
   - plan.affected_symbols Symbols involved
   - plan.execution_steps Ordered action steps
   - plan.reasoning       Analysis explanation

5. Serialize for downstream agents:
   plan_json = planner.plan_to_json(plan)
   # Use in Coder, Tester, Reviewer, Committer agents
